# Bisma Batool — Portfolio

A personal portfolio website showcasing my background, education, and skills as a programming student.

🔗 **Live Site:** https://bisma-batool.netlify.app/

## About

This is my personal portfolio site built to introduce myself, share my education background, highlight my technical skills, and provide a way for people to get in touch with me.

## Features

- Interactive animated background (particle canvas that reacts to mouse/touch movement)
- Smooth scroll navigation
- Animated skill progress bars that fill in when scrolled into view
- Responsive design (mobile-friendly)
- Contact form powered by [Formspree](https://formspree.io/) with live status feedback
- Direct contact links (Email & WhatsApp)

## Sections

- **Home** — Introduction
- **About** — A bit more about me
- **Education** — My academic background
- **Skills** — Frontend, Backend, and Tools I work with
- **Contact** — Get in touch via form, email, or WhatsApp

## Built With

- HTML5
- CSS3 (custom properties, grid, flexbox, responsive media queries)
- JavaScript (Canvas API, IntersectionObserver, Fetch API)
- [Formspree](https://formspree.io/) for form handling
- Fonts: [Fraunces](https://fonts.google.com/specimen/Fraunces), [Inter](https://fonts.google.com/specimen/Inter), [JetBrains Mono](https://fonts.google.com/specimen/JetBrains+Mono)

## Project Structure

```
├── index.html      # Main HTML structure
├── style.css       # All styling
├── script.js       # Interactivity (canvas animation, skill bars, contact form)

```
